# Swift Demo

A CLI example implementation of the JuiceboxSdk for Swift.

```
USAGE: demo --configuration <configuration> --auth-tokens <auth-tokens> --tls-certificate <tls-certificate>

OPTIONS:
  -c, --configuration <configuration>
                          The configuration for the client SDK, in JSON format
  -a, --auth-tokens <auth-tokens>
                          The auth tokens for the client SDK, as a JSON string
                          mapping realm ID to base64-encoded JWT
  -t, --tls-certificate <tls-certificate>
                          The path to the TLS certificate used by the realms
  -h, --help              Show help information.
```
